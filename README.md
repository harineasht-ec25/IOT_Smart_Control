# IOT_Smart_Control
🚀 ESP32-based smart relay control system using HiveMQ MQTT for real-time remote automation 🌍. Supports WiFi provisioning through captive portal 📶, 4-channel relay control ⚡, MQTT cloud communication ☁️, responsive web dashboard 💻, automatic reconnection 🔄, and global device access for IoT &amp; smart automation 🔥.
